# EVE: A Generator-Verifier System for Generative Policies

Project page for **EVE** (Embodied Verifier Ensembles) — a modular generator-verifier framework that
boosts pretrained generative robot policies at test time using zero-shot VLM-based verifier ensembles, with
no additional training.

- **Paper:** [arXiv:2512.21430](https://arxiv.org/abs/2512.21430)
